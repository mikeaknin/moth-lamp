# Privacy Policy — MOTH//LAMP

**Last updated:** 2026-07-09  
**Operator contact placeholder:** support@example.com  

## Summary

MOTH//LAMP is a browser game. Core play does not require an account or wallet.

## Data we store on your device

Using browser `localStorage`:

- Game settings (volumes, accessibility toggles, difficulty)  
- Local high scores  
- Tutorial completion flag  
- Optional local profile label (display name, optional wallet address if you link)

This data stays in your browser unless you clear site data.

## Wallet / Base Account (optional)

- Connecting a wallet is **optional** and **user-initiated**.  
- We do not prompt for a wallet on first load.  
- If you use Sign-In with Ethereum, you create a signature that proves address ownership; this build stores the address **locally** for labeling only.  
- We do not receive your private keys.  
- Wallet providers (e.g. Base Account) have their own privacy policies.

## Analytics

No third-party analytics SDK is bundled by default.

## Children

The game is a general audience action title. Do not provide personal information if you are not permitted to do so under local law.

## Changes

Policy text may be updated when features change. The in-app Privacy page and this file should stay aligned.
